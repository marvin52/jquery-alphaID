What is jQuery AlphaID ?

jQuery AlphaID is a simple plugin to create alpha-numeric ID's to your projects.

Let's say you have an URL like: 
http://myblog.com/post/3892761
With AlphaID plugin you can convert this to something like this: 
http://myblog.com/post/Fp0r

Examples

See it working

Example 1

The default event is keyup.

Your AlphaID: 

$('#ex-1-input').alphaID({target:'#ex-1-target'});

Really simple to use :)

Changing the event

Now only when we mouseover the input, the AlphaID will be available.

Your AlphaID: 

$('#ex-1-input').alphaID({target:'#ex-1-target'});

Changing global Index

It's pretty easy to change the global Index, but be careful of junk characters (like "#, %, &, @") and with the amount of characters that is directly linked to the amount of combinations.

$.fn.alphaID.defaults.index = 'my_new_hash';

To change the hash on a specific case, use: 

$('#ex-1-input').alphaID({target:'#ex-1-target',index:'temporary_index_here'});

Install

To make AlphaID work you only need to declare jQuery ( > v1.1x.x ), and alphaId.min.js.

<script src="http://code.jquery.com/jquery-1.11.1.min.js"></script>
<script src="alphaID.min.js"></script>
